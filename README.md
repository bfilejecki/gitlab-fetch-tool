# Gitlab Fetch Tool
## Running
### Create virtual environment
```
    python3 -m venv ./venv
```
### Install requirements
```
    ./venv/bin/pip3 install -r ./requirements.txt.
```
### Prepare config values using template file
```
    cp template-config.yml config.yml
```
### Run
```
    ./venv/bin/python3 main.py
```
